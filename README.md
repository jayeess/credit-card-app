# Credit Card Application

A modern credit card application wizard with elegant animations and smooth user experience.

## Features

- Multi-step form wizard with animated progress indicator
- Real-time form validation
- Drag & drop file upload for documents
- PostgreSQL database with Prisma ORM
- Fully responsive design
- Deployed on Vercel

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Prisma with PostgreSQL (Neon)
- Zod validation

## Getting Started

```bash
npm install
npx prisma db push
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000)
